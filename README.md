**In this study, we used machine learning (ML) based approach for the detection of GSD. 
Statistical analysis reveal multiple clinically relevant bioimpedance and biochemical markers, paving the way for the early detection of GSD. 
We believe this study will serve as a proof-of-concept on how ML models can be potentially leveraged for the detection of GSD at community level**

1. G**allStone_dataset-uci.csv** : Dataset used in the article.
2. S**kleran_GallStone.ipynb** : ML model development
3. **Shap_Analysis_GallStone.ipynb** : SHAP analysis to interpret ML model and identify important features
4. **Pvalue_GallStone.ipynb** : Statistical analysis for the features
